Overview

XSite uses Git to manage its source code.

Web Access

Git allows web access:

http://git.codehaus.org/gitweb.cgi?p=xsite-git.git

Anonymous Access

git clone git://git.codehaus.org/xsite-git.git

Committer Access

git clone ssh://git@git.codehaus.org/xsite-git.git

Committers will need to upload their public via Xircles.

Access through a proxy

Git support HTTP protocol for cloning, which normally is allowed in corporate firewalls

git clone http://git.codehaus.org/xsite-git.git

You may need to set http.proxy via git-config.
